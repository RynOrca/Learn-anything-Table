## 阶段一：基础入门

> 目标：掌握 Vue 的核心概念和基本语法，能创建简单的交互页面

### 1.1 声明式渲染
- 使用模板语法将数据绑定到 DOM，理解响应式数据的基本概念

### 1.2 指令系统
- 学习 v-bind、v-model、v-if、v-for 等常用指令的用法和作用

### 1.3 事件处理
- 使用 v-on 监听 DOM 事件，在方法中处理用户交互

### 1.4 计算属性
- 使用 computed 定义基于依赖数据缓存的派生值，提升性能

### 1.5 侦听器
- 使用 watch 监听数据变化并执行异步或复杂操作

### 1.6 条件渲染
- 使用 v-if/v-else-if/v-else 和 v-show 控制元素显示与隐藏
